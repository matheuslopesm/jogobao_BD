Criar banco: CREATE DATABASE nome_do_banco;
Deletar banco: DROP DATABASE nome_do_banco;
Listar bancos: \l
Conectar no banco de dados: \c nome_do_banco;

Verificar as tabelas existentes no banco: \dt;
Buscar dados da tabela: \d nome_da_tabela;

Criar tabela: 
CREATE TABLE nome_da_tabela(
    name VARCHAR (50) NOT NULL,
    star_name VARCHAR (50) NOT NULL,
    idade INT
    );

Deletar tabela:
DROP TABLE nome_da_tabela;

Alterar tabela: 
ALTER TABLE nome_da_tabela ADD COLUMN description text;
Aqui ele está alterando a tabela adicionando a coluna descrição de tipo texto.

ALTER TABLE nome_da_tabela DROP COLUMN description text;

Inserir dados nas linhas (ou seja, dados dos atributos):
INSERT INTO nome_da_tabela (name, star_name, idade) VALUES ('saturno', 'sol', '45', etc...);

Para ver os dados inseridos: 
SELECT * FROM nome_da_tabela;

Quer ver o dado de alguma coluna?
SELECT nome_da_coluna FROM nome_da_tabela;

Quer ver algum dado com condição?
SELECT nome_da_coluna FROM nome_da_tabela WHERE condição; (exemplo: SELECT nome_da_coluna FROM nome_da_tabela WHERE idade<4);

Atualizar tabela:
UPDATE nome_da_tabela SET star_name = 'Tempest' WHERE id = 1;

Deletar linha da tabela:
DELETE FROM nome_da_tabela WHERE condição;

Para inserir chave estrangeira, vc faria por exemplo:
CONSTRAINT nome_da_constraint FOREIGN KEY (nome_da_chave_estrangeira) REFERENCES nome_da_tabela(nome_do_campo)
